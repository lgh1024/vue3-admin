# Vue 3 + TypeScript + Vite + Element Plus

## 项目名称
- 电商后台管理系统

### 项目克隆
```
git clone https://github.com/lgh1024/vue3-admin.git
``` 
克隆太慢的话用下面的会快一点
```
git clone https://gitee.com/lgh1024/vue3-admin.git
```
#### 安装依赖
```
npm install
```
#### 项目运行
```
npm run serve
```

#### 项目打包
```
npm run build
```
